# TimeKeeper

生活習慣パターンとカレンダー予定を統合し、1日のタイムスケジュールを自動生成するWebアプリケーション

## プロジェクト概要

TimekeeperはApple Calendarと連携し、ユーザーの生活習慣パターンに基づいて最適な1日のスケジュールを自動生成します。移動時間の自動計算、リアルタイムの進捗追跡、遅延時の自動調整機能を備えています。

## 主な機能

### ✅ MVP機能

1. **生活習慣パターン管理**
   - 複数のパターン作成（平日、休日、リモートワークなど）
   - カスタム習慣項目の追加（朝食、運動、勉強、休憩など）
   - ルールベース自動選択（曜日・キーワードマッチング）

2. **カレンダー連携**
   - Apple Calendar（CalDAV API）との完全自動連携
   - 予定の自動取得・同期
   - 場所情報の自動マッチング

3. **移動時間管理**
   - 場所マスタ管理（自宅、会社、ジムなど）
   - 移動ルート・所要時間の登録
   - 複数移動手段対応（徒歩、車、電車など）
   - 場所変更時の移動イベント自動挿入

4. **自動スケジューリング**
   - 生活習慣 + カレンダー予定の統合
   - 現在地追跡による移動時間の動的計算
   - 1日のタイムライン自動生成

5. **リアルタイム進捗追跡**
   - イベント完了チェック機能
   - 次のイベントまでのカウントダウン
   - 遅延検出・自動調整
   - 移動手段変更の提案

### 🚀 将来的な機能（v2以降）

- データのクラウド同期（Supabase）
- AI学習による最適化提案
- 通知機能
- 複数カレンダー対応
- チーム共有機能

## 使い方

### 基本的な使用フロー

#### 1. 初期設定
```
場所の登録 → 移動ルートの設定 → 習慣項目の作成 → パターンの作成
```

#### 2. 日常的な使い方
```
朝起きる
↓
アプリでスケジュール確認（自動生成済み）
↓
イベント実行 & 完了チェック
↓
遅延があれば自動調整
↓
次のイベントまでの時間を確認
```
