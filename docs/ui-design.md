# UI設計書

## デザインコンセプト

- **シンプル・ミニマル**: 余計な装飾を排除し、情報を見やすく
- **タイムライン中心**: 1日の流れを直感的に把握
- **レスポンシブ**: モバイルファーストで設計
- **アクセシビリティ**: shadcn/uiのアクセシブルなコンポーネント使用

## 共通UIコンポーネント

### 1. TimelineItem

```typescript
<TimelineItem
  event={event}
  isActive={event.status === 'active'}
  isCompleted={event.status === 'completed'}
  onComplete={() => handleComplete(event.id)}
  onSkip={() => handleSkip(event.id)}
/>
```

**表示内容**:
- 時刻（adjustedStartTime）
- アイコン
- タイトル
- 所要時間
- 場所
- ステータス（✅完了、🔵進行中、⏰予定、⚠️警告）
- アクション（完了ボタン、スキップボタン）

### 2. PatternSelector

```typescript
<PatternSelector
  currentPattern={pattern}
  patterns={allPatterns}
  onChange={(patternId) => handlePatternChange(patternId)}
/>
```

### 3. LocationBadge

```typescript
<LocationBadge
  locationId={locationId}
  showIcon={true}
/>
```

### 4. TravelMethodIcon

```typescript
<TravelMethodIcon method="car" />
// 🚗, 🚶, 🚃, 🚌, 🚴
```

### 5. ProgressBar

```typescript
<ProgressBar
  current={currentMinutes}
  total={totalMinutes}
  color="blue"
/>
```

### 6. DelayNotification

遅延警告の通知カード

```typescript
<DelayNotification
  delay={delayRecord}
  suggestions={suggestions}
  onAccept={handleAcceptSuggestion}
  onDismiss={handleDismiss}
/>
```

## レスポンシブデザイン

### ブレークポイント

```typescript

  sm: '640px',   // スマホ横
```

- タッチターゲットは最低44x44px

- サイドバーナビゲーション
